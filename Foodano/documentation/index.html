Fonts

Google free fonts:

  • Body Font : Lato
  • Title Font : Advent Pro & Amatic SC

On the header part of every page, you will find the HTML tag of fonts. If you want to change the fonts just copy your HTML embed code and replace it with the current one. Than Open assets/css/style.css file and change the fonts name from typography area. That's it...
